# AI Assistant UI - General Project Information

This project is the User Interface (UI) for our AI Assistant application. And the backend is on path: D:\proj\ai-chatbot\src

The core features and capabilities of this project include:

1. **Normal Chat**: Provides a standard conversational interface for users to chat with the AI.
2. **RAG Integration (Retrieval-Augmented Generation)**: Applies RAG to the system to provide intelligent recommendations and context-aware responses.
3. **Social Network Connectivity**: Connects to various social networks to facilitate buying, chatting, and automation directly through the interface.
4. **Private Chatbot Settings**: Includes customizable settings allowing us to tailor the private chatbot to our specific ideas and requirements.
5. **Mobile Responsive**: The UI is designed to be responsive and work seamlessly on mobile devices.
6. **Language Support**: The UI supports multiple languages including English, Arabic, and French. Default Language is Arabic
7. **Theme Support**: The UI supports multiple themes including Light and Dark.
8. **Model Info** This app uses Gemini models to provide its services
9. **Chat History**: The UI supports chat history, which allows us to view our previous conversations.
10. **Auto Save**: The UI supports auto save, which automatically saves our conversations.
11. **Customization**: The UI supports customization, which allows us to customize the UI to our specific ideas and requirements.
12. **RTL Support**: The UI supports right-to-left languages, which is important for Arabic and French.
13. **File Upload**: The UI supports file upload, which allows us to upload files to the server.
14. **Context Menu**: The UI supports context menu, which allows us to perform actions on the chat messages.
15. **Notifications**: The UI supports notifications, which allows us to receive notifications about the chat messages.
16. **Voice Input**: The UI supports voice input, which allows us to speak to the AI.
17. **Speech Recognition**: The UI supports speech recognition, which allows us to convert speech to text.
18. **Text-to-Speech**: The UI supports text-to-speech, which allows us to convert text to speech.
19. **Web Search**: The UI supports web search, which allows us to search the web for information.
20. **Google Search**: The UI supports Google search, which allows us to search Google for information.
21. **Bing Search**: The UI supports Bing search, which allows us to search Bing for information.
22. **Youtube Search**: The UI supports Youtube search, which allows us to search Youtube for information.
