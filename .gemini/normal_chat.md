# AI Assistant UI - Normal Chat Skill

This skill outlines the capabilities and configuration for the "Normal Chat" feature in the AI Assistant UI.

## Core Features

1. **Intelligent Chat Interface**: 
   - Delivers a smart conversational experience, enabling the AI to understand and respond to user queries naturally and effectively.
   - Designed to handle general inquiries, brainstorming, and day-to-day assistance.

2. **Customizable Context Setup**: 
   - Allows users to define and configure specific contexts or personas for the conversation. 
   - Users can set up rules, background information, or desired tones that the AI should adhere to during the chat session.

3. **User Interaction**: 
   - Provides a responsive and intuitive chat UI.
   - Should support standard messaging features to ensure a seamless conversational flow.
